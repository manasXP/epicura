# Gitignore Templates

Template `.gitignore` files for each Epicura repository. Copy the relevant file to the repo root as `.gitignore` when scaffolding.

| Template | Repository | Stack |
|----------|-----------|-------|
| [[epicura-firmware.gitignore]] | `epicura-firmware` | C / STM32 / FreeRTOS |
| [[epicura-cm5.gitignore]] | `epicura-cm5` | Python / Docker / Yocto |
| [[epicura-api.gitignore]] | `epicura-api` | TypeScript / Node / Next.js |
| [[epicura-ios.gitignore]] | `epicura-ios` | Swift / Xcode |
| [[epicura-android.gitignore]] | `epicura-android` | Kotlin / Gradle |

> [!note] The root `epicura` docs repo already has its own `.gitignore` with KiCAD and Obsidian entries.
